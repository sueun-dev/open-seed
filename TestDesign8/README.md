# 🐍 Neon Snake

A classic Snake game with neon visuals, progressive difficulty, and synthesized sound effects.

## Play

Open `index.html` in any modern browser. No server or build step required.

## Controls

| Key | Action |
|---|---|
| Arrow Keys / WASD | Move snake |
| P | Pause / Resume |
| Space / Enter | Start / Restart |

## Features

- **HTML5 Canvas** rendering with neon glow effects
- **Progressive difficulty** — speed increases every 100 points
- **Web Audio API** synthesized sound effects (eat, die, level up)
- **localStorage** high score persistence
- **Keyboard controls** — Arrow keys and WASD
- **Game screens** — Start, Pause, Game Over
- **Particle effects** on food collection and death
- **Zero dependencies** — single self-contained HTML file
