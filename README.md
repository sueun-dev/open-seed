## What is Open Seed?

Open Seed is an **autonomous AI agent** that builds complete software from a single prompt. It does not autocomplete. It does not suggest. It **plans, writes, tests, debugs, and ships** вҖ” fully autonomously.

```
$ openseed run "todo м•ұ л§Ңл“Өм–ҙмӨҳ"

вҶ’ Phase 1: ANALYZE вҖ” intent classification, codebase assessment
вҶ’ Phase 2: DESIGN вҖ” architecture, file structure, task breakdown
вҶ’ Phase 3: BUILD вҖ” 40 specialist roles write complete code
вҶ’ Phase 4: VERIFY вҖ” type-check, tests, build validation
вҶ’ Phase 5: IMPROVE вҖ” security audit, performance, test coverage
вҶ’ Phase 6: REVIEW вҖ” final quality gate

вң“ 6/6 phases passed
рҹ“Ғ workspace/todo-app/ вҖ” complete runnable project
```

Powered by **GPT-5.4 + Claude Opus 4.6** via OAuth. **$0 cost** with subscription.

## Quick Start

### Web UI (Recommended)

```bash
git clone https://github.com/sueun-dev/open-seed.git
cd open-seed
npm install
npm run build
node app/server.js --port 4040
```

Open **http://localhost:4040** вҖ” full IDE with AGI mode, explorer, editor, terminal, AI chat.

### CLI

```bash
# Single agent mode
node dist/cli.js run "Create a calculator with add, subtract, multiply, divide"

# Team mode (parallel specialist workers)
node dist/cli.js team "Build a REST API with authentication"

# One-prompt app generation
node dist/cli.js create "Build a todo app with React"

# Diagnostics
node dist/cli.js doctor
```

### AGI Mode (Web UI)

Click **AGI Mode** вҶ’ Type your prompt вҶ’ Click **AGI Start**

The engine automatically runs all 6 phases. Generated projects are saved to `workspace/{project-name}/` with complete structure:

```

в”ңв”Җв”Җ package.json          # dependencies, scripts
```
